# 🚗 Vehicle Sales Data Analysis (EDA)

## 📊 Project Overview
This project focuses on Exploratory Data Analysis (EDA) of vehicle sales data using Excel.  
The goal is to analyze sales performance, customer trends, and business insights through data cleaning, data management, and visualization.

---

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Graphs  
- Data Cleaning Techniques  

---

## 🔍 Data Cleaning & Management
- Removed missing and duplicate values  
- Standardized column names and formats  
- Handled inconsistent data entries  
- Organized raw data into structured format  
- Created calculated fields for better analysis  

---

## 📈 Key Analysis Performed
- Sales analysis by country  
- Sales distribution by deal size  
- Product category performance  
- Order status analysis (Shipped, Cancelled, etc.)  
- Year-wise sales trends  

---

## 📊 Dashboard Features
- Interactive filters (Country, City, Month)  
- KPI-based insights  
- Multiple charts for comparison  
- Easy-to-understand visual representation  

---

## 💡 Key Insights
- Certain countries generate higher average sales  
- Large deal sizes contribute significantly to revenue  
- Majority of orders are successfully shipped  
- Sales trends show growth over the years  

---

## 🚀 Conclusion
This project demonstrates the use of Excel for data cleaning, data management, and dashboard creation to generate meaningful business insights.
